Trivia: Verbo "To Be"

Demuestra cuánto sabes sobre el verbo "to be" en inglés.

1. ¿Cuál es la forma correcta para "I"?

2. ¿Cuál es la forma correcta para "they"?

3. Traduce: "She is happy"

4. ¿Cuál es la forma correcta para "he"?

5. ¿Cuál es la forma correcta para "we"?

6. Traduce: "You are students"

7. ¿Cuál es la forma correcta para "it"?

8. Traduce: "They are friends"

9. ¿Cuál es la forma correcta para "you" (singular)?

10. Traduce: "We are teachers"

11. ¿Cuál es la forma correcta para "you" (plural)?

12. Traduce: "He is my brother"

13. ¿Cuál es la forma negativa de "I am"?

14. ¿Cuál es la contracción de "they are"?

15. Traduce: "It is cold"
